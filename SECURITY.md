# Security

## Reporting a vulnerability

Please report security issues responsibly:

- **Preferred:** [GitHub Security Advisories](https://github.com/AkshantVats/infra-ai-streaming/security/advisories/new) (private disclosure on this repository).
- **Email:** akshant3@gmail.com (include steps to reproduce and impact if known).

Do not open public issues for undisclosed security bugs.

We will acknowledge reports as soon as practical and coordinate fixes and disclosure with you.
